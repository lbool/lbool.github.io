Github Pages绑定域名

发表于 2016-01-27 | 分类于 笔记

背景

现在好多非IT人员都拥有了自己的博客，无奈服务器租金略贵且不稳定，许多人投向了Github的怀抱，去整个静态博客一挂，啥事儿没有，完美！本文就简单讲下Github Pages如何绑定自己的域名。

创建博客

在github创建博客用的是github page

github pages是一个免费的静态网站托管平台，由github提供，它具有以下特点：

  1. 免空间费(据说只有300MB)，免流量费
  2. 能装逼，全世界通装，24K纯金
  3. 具有项目主页和个人主页两种选择
  4. 支持页面生成，可以使用jekyll或Hexo来布局页面，使用markdown来书写正文
  5. 可以自定义域名

它其实就是在github新建一个特殊的仓库，特殊在哪里呢，这个仓库名必须是固定格式的，比如你github名是shabi，那么你这个仓库名必须是 shabi.github.io。
如果你不会新建的话，要么百度，google，要么放弃吧

绑定域名

  1. 在自己网站项目repo的根目录添加CNAME，里面的内容为域名不要http以及www等前缀，只需写入域名本身，例如
shabi.com
  1. 添加域名设置两个A记录，分别是@和www，ip地址填
192.30.252.153
192.30.252.154

这个时候生效的话你ping你的域名就应该能粗出来了
访问试试，开始尽情的装逼吧
